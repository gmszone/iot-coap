#物联网系统 CoAP版

这是一个开源的最小物联网系统的CoAP版，如果你还是一个初学者建议用HTTP版 [https://github.com/gmszone/iot][iot]

##依赖库

 - RESTify
 - Node-CoAP
 - Node Sqlite3

##install

需要安装有Sqlite3及Nodejs


###安装node依赖

     npm install

###运行

     node server.js

###Test

    coap coap://localhost

####IOT-MQTT#####

1.安装依赖库

    npm install mqtt
    npm install mosca bunyan -g

2.启动mosca

    mosca -v | bunyan

3.测试mqtt-client

    node mqtt-subscribe-client.js
    node mqtt-publish-client.js
